# Proyecto Unicornios + Productos

## Descripción
Aplicación web en React para gestionar dos módulos independientes:
- **Unicornios**: CRUD de unicornios con validaciones usando Formik + Yup.
- **Productos**: CRUD de productos, funcionando igual que unicornios con Context API.

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/MauriTorres9/unicornios-productos.git
cd unicornios-productos
Instalar dependencias:

bash
Copiar código
npm install
Ejecutar la aplicación:

bash
Copiar código
npm start
Abrir en el navegador: http://localhost:3000

Uso
/unicornios → Listado de unicornios, crear, editar, eliminar.

/productos → Listado de productos, crear, editar, eliminar.
